== PostgreSQL Weekly News - 20 novembre 2005 ==

Par Jean-Paul Argudo le lundi 21 novembre 2005, 08:56 - PostgreSQL Weekly News - Lien permanent

Alvaro Herrera a soumis un patch qui ajoute deux nouvelles commandes: DROP OWNED et REASSIGN OWNED BY dans la version CVS.

Pour sa part, Andrew Dunstan a ajouté un patch DROP IF EXISTS dans le CVS. Cela devrait aider le travail avec les rôles et adoucir les migrations depuis les autres SGBDR ainsi que le travail avec les scripts DDL (NDT: Data Definition Language).

Tom Lane et AL. ont ajouté le support des NULL dans les tableaux, dans la version CVS. Andrew Dunstan a ajouté l'équivalent dans PL/Perl.

Il y a eu de grandes discussions en vue de l'amélioration du COUNT(*) pour les grosses tables.

http://www.blastwave.org/
