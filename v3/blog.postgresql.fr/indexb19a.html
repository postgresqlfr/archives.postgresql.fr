Nouvelles hebdomadaires de PostgreSQL - 5 Décembre 2005

Quelques discussions au sujet de l'implémentation du type NUMERIC sont apparues sur la liste -hackers. La proposition de le limiter à 508 signes semble être la voie qui a été choisie.

PostgreSQL 8.1.1 sortira cette semaine.

Nouvelles des produits autour de PostgreSQL
  • MySQL Compatibility Project a lancé sa première version beta : http://pgfoundry.org/projects/mysqlcompat/
  • PHP 5.1 inclut maintenant plusieurs des nouvelles fonctions de la version 3 du protocole de PostgreSQL : http://people.planetpostgresql.org/devrim/index.php?/archives/24-New-PostgreSQL-functions-in-PHP-5.1.html
  • Debian dispose maintenant d'un paquetage officiel pour Slony-I : http://packages.qa.debian.org/s/slony1.html
  • MicroOLAP met à disposition un nouveau programme en version beta pour sont Database Designer for PostgreSQL http://microolap.com/my/beta/
  • PgBrowse 1.3 est sorti http://homepage.mac.com/levanj/TclTk
  • pqa 1.6 est sorti http://pgfoundry.org/projects/pqa/
  • phpPgAdmin 4.0.1 vient de sortir avec de nombreux correctifs de bugs http://phppgadmin.sf.net/
  • Greenplum passe en version Beta2 pour Bizgres MPP http://www.greenplum.com/mpp-beta/

Evènement locaux autour de PostgreSQL

Devrim Gunduz et Volkan Yazycy animeront un séminaire au sujet des nouvelles fonctionnalités de PostgreSQL 8.1 le 11 décembre 2005 à Istambul : http://www.postgresql.org/about/event.326

PostgreSQL dans la presse

  • FeedLounge est passé de MySQL à PostgreSQL. En voici les raisons http://feedlounge.com/blog/2005/11/20/switched-to-postgresql/
  • Chris Campbell du Big Nerd Ranch démontre comment abaisser les priorité des requêtes. http://weblog.bignerdranch.com/?p=11
  • Planet PostgreSQL: http://www.planetpostgresql.org/
  • General Bits, est publié tous les lundis (PST/PDT): http://www.varlena.com/GeneralBits/ (not-so) Simple Selects and Complex Aggregates
