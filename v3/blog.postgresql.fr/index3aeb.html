Sortie des versions 7.3.12, 7.4.10, 8.0.5 et 8.1.1

Par gleu le mardi 13 décembre 2005, 12:55 - Dans les bacs - Lien permanent

Les développeurs de PostgreSQL viennent de sortir des versions correctives pour toutes les versions majeures actuellement maintenues. Pour une liste complète des bogues corrigés, merci de lire le fichier HISTORY approprié disponible dans chaque archive tar.

Les sources sont disponibles aux URL suivantes :

  • version 7.3.12 ;
  • version 7.4.10 ;
  • version 8.0.5 ;
  • version 8.1.1.

Des binaires sont disponibles pour Linux :

  • version 7.3.12 ;
  • version 7.4.10 ;
  • version 8.0.5 ;
  • version 8.1.1.

Des binaires sont aussi disponibles pour Windows.

Les versions correctives ne nécessitent pas, généralement de sauvegarde/restauration lorsqu'il n'y a pas de saut de version majeure. Vérifiez néanmoins dans les procédures de mise à jour.

Les documentations des différentes versions ont aussi été mises à jour en français :

  • version 7.4.10 ;
  • version 8.0.5 ;
  • version 8.1.1.
