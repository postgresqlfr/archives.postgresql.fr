Nouvelles Hebdomadaires de PostgreSQL - 16 avril 2006

Dan Harris a dit : Encore une fois, merci à tous les développeurs pour votre dur travail sur PostgreSQL. C'est de loin la plus agréable expérience de gestion de base de données avec laquelle j'ai pu travailler. : http://archives.postgresql.org/pgsql-performance/2006-04/msg00247.php

Nous avons besoin d'étudiants et de mentors qui souhaitent travailler sur des projets PostgreSQL pour le prochain Summer Of Code de Google. Un des gagnants de l'année dernière a été Query By Example. Si vous êtes enseignant ou étudiant interressé par un projet orienté base de données, et plus spécialement, si vous avez déjà votre propre projet en relation avec PostgreSQL, veuillez contacter Josh Berkus (josh postgresql.org) dès que possible.

Tom Lane à réécrit CREATE AGGREGATE, ouvrant la porte à des aggrégats à entrées multiples.

Les inscriptions pour l'anniversaire de PostgreSQL sont ouvertes : http://conference.postgresql.org/Registration.

L'anniversaire de PostgreSQL est sponsorisé par Afilias, EnterpriseDB, GreenPlum, Pervasive et Sun Microsystems.
