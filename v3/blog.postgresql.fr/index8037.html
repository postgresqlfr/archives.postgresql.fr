Un postgresql.conf complètement commenté !

Par Jean-Paul Argudo le jeudi 10 février 2005, 09:22 - Événements - Lien permanent

Josh Berkus et Joe Conway se sont lancés ce début d'année dans la rédaction d'un livre sur la dernière version de PostgreSQL. Sur leur site, une sorte de blog, ils viennent de poster un document plus qu'intéressant. Il s'agit d'un fichier postgresql.conf complètement commenté !

Je vous engage donc à lire attentivement ce document qui présente pour chaque clé du postgresql.conf :

  • son nom
  • les valeurs possibles
  • la valeur par défaut
  • si la variable peut être définie à chaud ou à froid
  • si un raccourci existe pour pg_ctl
  • ce que la documentation officielle dit sur la variable
  • le commentaire des deux auteurs, tout à fait précieux !

Deux formats sont possibles pour cette documentation :

  • format HTML
  • format PDF

Bonne lecture !
