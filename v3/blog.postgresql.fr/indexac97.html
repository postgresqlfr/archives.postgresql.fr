Nouvelles hebdomadaires de PostgreSQL - 5 Septembre 2005

Il y a eu une discussion animée sur le support du tri alphabétique en fonction de la locale utilisée. Il apparaît clairement que cette fonctionnalité serait largement traîtée dans la version 8.2 de PostgreSQL.

SCALE 4x, le Southern California Linux Expo fait un appel à candidature pour les présentations. Voir ci-après pour plus d'informations. http://www.socallinuxexpo.org

Mark Wong d'OSDL a débuté le travail sur dbt-4, un implémentation correcte de TPC-App. TPC-App est un outil de charge de services web b2b et Mark aimerait avoir un peu plus d'aide de la part de développeurs Java.

Il y a un grand nombre de nouveaux développeurs sur la page « developers' bio » http://www.postgresql.org/developer/bios

Nouvelles des produits autour de PostgreSQL

  • Nouvelle version de EMS PostgreSQL Manager 3.2 pour Windows. http://www.sqlmanager.net/products/postgresql/manager/
  • Nouvelle version de PostgresDAC 2.2.1. http://www.microolap.com/products/connectivity/postgresdac/
  • Sortie de pgPool 2.6.3 http://pgfoundry.org/projects/pgpool/
  • C'est parti pour une nouvelle implémentation du « back-end » PostgreSQL pour Bugzillahttps://bugzilla.mozilla.org/show_bug.cgi?id=98304
  • PostgreSQL RPM Building Project ( http://pgfoundry.org/projects/pgsqlrpms ) a permis de fabriquer les RPMs pour Red Hat Linux 9, Red Hat Enterprise Linux 3.0 et 4. D'autres viendront par la suite. http://developer.postgresql.org/~devrim/rpms/8.1/beta1/rpms/
