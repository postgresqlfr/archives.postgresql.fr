Par jca le mercredi 21 décembre 2005, 08:46 - PostgreSQL Weekly News - Lien permanent

Après quelques semaines d'absence, la traduction française du PostgreSQL Weekly News revient. Je prie les lecteurs d'accepter toutes mes excuses pour cette interruption de traduction

Nouvelles hebdomadaires de PostgreSQL - 18 Décembre 2005

Les versions 8.1.1, 8.0.5, 7.4.10 et 7.3.12 sont sorties. Mettez à jour vos versions avec ces dernières afin de bénéficier de la correction de tous les bugs.

Nouvelles des produits autour de PostgreSQL
  • abaXX.components 4.6.2 est sorti http://www.abaxx.com/releaseinfo/
  • pgAdmin v1.4.1 est sorti http://www.postgresql.org/ftp/pgadmin3/release/v1.4.1/
  • Les RPMs pour les versions 8.1.1, 8.0.5, 7.4.10 et 7.3.12 sont disponibles. http://www.postgresql.org/ftp/binary/ http://www.postgresql.org/download/bittorrent
  • ExtenDB a sorti une version de Standard Edition avec le support de PostgreSQL 8.1. http://www.extendb.com/pr81.php
  • PgBrowse 1.4 est sorti. http://homepage.mac.com/levanj/TclTk/
  • kbforge sait maintenant utiliser PostgreSQL http://kbforge.com/
  • Wikipgedia-0.0.3 est sorti, incluant des corrections pour la sécurité. http://pgfoundry.org/projects/wikipedia/

Evènement locaux autour de PostgreSQL

Cette semaine, tous les évènements ont été globaux.

PostgreSQL dans la presse

  • Joe Brockmeier de Newsforge a interviewé Christopher Kings-Lynne et Josh Berkus sur le projet MySQL Compatibility. http://software.newsforge.com/software/05/12/15/1611251.shtml?tid=72
  • Larry Rosenman a rejoint Pervasive Software qui supporte PostgreSQL. Ce nouveau travail lui permettra d'avoir plus de temps pour travailler sur la version de la communauté.
  • Planet PostgreSQL: http://www.planetpostgresql.org/
  • General Bits, publié tous les lundis (PST/PDT): http://www.varlena.com/GeneralBits/

Les Nouvelles hebdomadaires de PostgreSQL a été rédigé par David Fetter cette semaine.
