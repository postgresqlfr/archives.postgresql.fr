PostgreSQL 8.0.0 Release Candidate 5

Par jca le mercredi 12 janvier 2005, 14:08 - Dans les bacs - Lien permanent

Dans son annonce du 11 janvier 2005 sur la liste pg-announce, Marc G. Fournier nous apprenait :

Suite à la découverte de plusieurs petits bugs et d'un plus gros dans la version candidate 4, nous avons été obligés de produire une cinquième (et espérée dernière) version candidate de manière à pouvoir faire proprement les tests sur les changements avant la version finale.

Une liste des plates-formes connues peut être récupérée sur :

http://developer.postgresql.org/supported-platforms.html

Comme toujours nous cherchons à étendre cette liste, nous encourageons donc toute personne disposant d'une des plates-formes non listées de bien vouloir rapporter le succès ou l'échec d'une installation et d'une utilisation d'une version candidate 5.

A condition qu'il n'y ait pas de modification de code (documentation != code) dans le courant de la semaine prochaine, nous espérons que ce sera la dernière version candidate avant la version finale, en se fixant pour objectif le 19 (ou plus tôt).

Comme toujours, cette version est disponible sur tous les miroirs listés sur:

http://wwwmaster.postgresql.org/download/mirrors-ftp

Pour celles et ceux qui utilisent Bittorent, David Fetter a mis à jour les .torrents disponibles sur :

http://bt.postgresql.org

Prière de rapporter tout bug avec cette version candidate à :

pgsql-bugs@postgresql.org
