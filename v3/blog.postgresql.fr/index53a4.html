Installer PostgreSQL sous Windows, en natif, sans Cygwin? Facile!!!

Par Jean-Paul Argudo le lundi 30 août 2004, 12:28 - Articles - Lien permanent

Eh oui...

Avec PostgreSQL 8.0 qui est en béta 1, pour l'instant, rien de plus simple... Il n'y a qu'à aller faire un tour sur la page de PGInstaller!

Comme on vous le dit sur cette page, il vous faudra récupérer les exécutables sur sa page sur PGFoundry. Après il suffit de savoir cliquer! ;-)

Clic... clic... clic...

Enjoy!

PS: Vous aurez la version béta 1 de PG 8.0 d'installée sur votre machine Windows. Pour avoir la version stable, il vous faudra attendre un peu, que le cycle de béta test soit terminé... Si l'attente est trop longue: contribuez! La méthode la plus efficace pour que le cycle de béta test avance c'est de faire un report de bug
