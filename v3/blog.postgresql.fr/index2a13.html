PostgreSQL 8.2 beta3 et PgAdmin3 1.6 RC2

Par SAS le vendredi 10 novembre 2006, 09:39 - Dans les bacs - Lien permanent

Josh Berkus a annoncé ce jour :

PostgreSQL version 8.2, beta 3 est maintenant disponible pour test : http://www.postgresql.org/developer/beta

Cette version beta inclut un correctif conséquent d'un problème WAL : http://archives.postgresql.org/pgsql-hackers/2006-11/msg00117.php Les utilisateurs sont invités à tester la beta3 avec les PITR et d'essayer de simuler une panne de courant.

N'oubliez pas que la version 8.2 ne sortira que lorsque nous l'estimerons stable. Vos tests nous aident à accélérer le processus.

Dave Page, quant à lui annonce :

Je suis heureux d'annoncer la disponibilité de la seconde Relase Candidate de pgAdmin III v1.6.0. Les sources et les versions Windows, Slackware et Mac OSX sont téléchargeables sur : http://www.pgadmin.org/download/

Cette version contient de nombreux correctifs de bogues relevés dans les versions beta.

Cette version pre-stable peut contenir des bogues. Merci de l'essayer, à vos risque et périls, et de rapporter les bogues à pgadmin-support@postgresql.org.

Un grand nombre d'améliorations ont été apportées à pgAdmin 1.6. La liste complète est dans le changelog : http://www.pgadmin.org/development/changelog.php
