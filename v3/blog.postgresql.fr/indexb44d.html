PostgreSQL 8 remporte le prestigieux «&nbsp;Linux New Media Award&nbsp;» pour 2004!

Par Jean-Paul Argudo le lundi 8 novembre 2004, 15:13 - Événements - Lien permanent

PostgreSQL 8 remporte le prestigieux « Linux New Media Award » pour 2004!

Résultats en détail sur la page de Linux New Media Award 2004

Les nominés pour la meilleure base de données cette année étaient prestigieux (voir la liste originale):

  • MaxDB 7.5
  • Oracle 10g
  • MySQL 5.0
  • PostgreSQL 8
  • DB2
  • Apache Derby (Cloudscape)
  • Sybase Adaptive Server

Les résultats:

  • n°1: PostgreSQL 8 avec 48,7% des voix
  • n°2: MySQL 5.0 avec 37,0% des voix
  • n°3: Oracle 10g avec 5,9 % des voix

Toute l'équipe de PostgreSQLFr.org se félicite de cette victoire! Tous nos remerciements vont bien sûr aux développeurs de PostgreSQL et à toutes les personnes qui oeuvrent au quotidien pour ce projet (béta-testeurs, traducteurs, reports d'anomalie, etc.)...
