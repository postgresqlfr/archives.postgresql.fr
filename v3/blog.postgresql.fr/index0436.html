Benetl 1.2 : ETL gratuit pour postgreSQL

Par Benetl2008 le lundi 4 février 2008, 19:20 - Dans les bacs

  • benetl
  • ETL

La version 1.2 de Benetl est sortie.

Elle est disponible gratuitement sur : http://www.benetl.net

Elle apporte de nouvelles fonctionnalités, notamment la possibilité de générer des scripts d'insertion en base sans s'appuyer sur le MCD propre de Benetl (il est possible de choisir le nom d'une table). Cela fait de Benetl un loader évolué pour postgreSQL.
