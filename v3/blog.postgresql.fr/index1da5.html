PostgreSQL dans Linux Magazine France

Par Sebastien Lardiere le dimanche 9 mars 2008, 21:56 - Événements

PostgreSQL 8.3 fait la une du mensuel Linux Magazine France.

La nouvelle version majeure de PostgreSQL fait l'objet d'un article complet, recensant toutes les nouveautés.

Des nouvelles fonctionnalités pour les développeurs, jusqu'aux nouvelles directives de configuration, Guillaume Lelarge nous présente l'essentiel de ce qu'il faut savoir, afin d'exploiter au mieux PostgreSQL 8.3

Et pour ceux qui veulent en savoir encore plus, la documentation complète de PostgreSQL 8.3 est disponible, en français, sur le site docs.postgresqlfr.org.

Benetl 1.2 : ETL gratuit pour postgreSQL

Par Benetl2008 le lundi 4 février 2008, 19:20 - Dans les bacs

  • benetl
  • ETL

La version 1.2 de Benetl est sortie.

Elle est disponible gratuitement sur : http://www.benetl.net

Elle apporte de nouvelles fonctionnalités, notamment la possibilité de générer des scripts d'insertion en base sans s'appuyer sur le MCD propre de Benetl (il est possible de choisir le nom d'une table). Cela fait de Benetl un loader évolué pour postgreSQL.
