Nouvelles hebdomadaires de PostgreSQL - October 02 2005

Une discussion animée s'est poursuivie sur -hackers au sujet du tri externe, un point important à prendre en considération concernant les performances des grosses bases de données. Si la discussion est bonne, le code sera également bon.

Nouvelles des produits autour de PostgreSQL
  • Slony-I 1.1.1 est sorti http://slony.info/ Les fichiers RPMs de Slony-I 1.1.1 sont accessibles sur http://developer.postgresql.org/~devrim/slony
  • Le projet PostgresPy a sorti la version 0.1 de Proboscis http://pgfoundry.org/forum/forum.php?forum_id=522
  • iGroupCal supporte maintenant PostgreSQL http://www.igroupcal.com/
  • EMS a sorti la version 3.2 de EMS PostgreSQL Manager http://sqlmanager.net/en/products/postgresql/manager

PostgreSQL dans la presse

  • Grega Bremec a effectué une présentation à IBLOC 2005 en Slovaque. Les supports, également disponibles en Anglais, sont disponibles sur http://p0f.net/pub
    • Planet PostgreSQL: http://www.planetpostgresql.org/
  • General Bits: http://www.varlena.com/GeneralBits/ « Prendre en considération le cache »
