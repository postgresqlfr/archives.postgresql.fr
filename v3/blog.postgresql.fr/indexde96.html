Nouvelles neuves de PostgreSQL

Par chl le jeudi 30 octobre 2008, 23:27 - PostgreSQL Weekly News - Lien permanent

Nouvelles hebdomadaires de PostgreSQL - 19 octobre 2008

  • Simon Riggs a publié 2 versions d'une implémentation complète de "hot-standby".
  • Le premier PGDay européen fut un succès, avec près de 190 inscrits. Bernd Helmle a annoncé la publication en l'état de ses patchs pour les vues modifiables dans le courant de la semaine ; de même pour Zdenek Kotala qui fournira via git ses patchs permettant la MAJ de PostgreSQL "en place" [ndt: sans passer par l'étape pg_dump - reload]
  • Emmanuel Cecchet a publié quelques indications sur l'utilisation d'Eclipse pour développer PostgreSQL : http://wiki.postgresql.org/wiki/Working_with_Eclipse

Les nouveautés des produits dérivés

  • Archiveopteryx 3.0.2 : http://www.archiveopteryx.org/3.0.2

Offres d'emplois autour de PostgreSQL en octobre

http://archives.postgresql.org/pgsql-jobs/2008-10/threads.php

PostgreSQL Local

  • Dickson Guedes recherche des volontaires pour aider au "PgMeeting" à Florianópolis. Écrivez à guediz AT gmail DOT com.
  • PostgreSQL aura une table au LinuxLive (Olympia de Londres) du 23 au 25 octobre 2008. Écrivez à Dave Page pour participer : dpage AT pgadmin DOT org
  • Un RV PostgreSQL est programmé pour la "Linux fest" à Ontario : http://www.onlinux.ca/

PostgreSQL dans les média

  • Planet PostgreSQL : http://www.planetpostgresql.org/
