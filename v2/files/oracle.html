Oracle Announces the Acquisition of Open Source Software project, PostgreSQL 
Oracle Plans to Increase Support for Open Source Software  
REDWOOD SHORES, Calif.,   01-APRIL-2006  Oracle Corporation today announced the acquisition of global project PostgreSQL. PostgreSQL is distributed under an open source license. "Oracle has long been a supporter of open source software such as Linux and Apache," said Charles Rozwat, Oracle's Executive Vice President in charge of Database and Middleware Technology. "PostgreSQL is an innovative small project that develops open source database technology. Oracle intends to continue developing the PostgreSQL technology and expand our commitment to open source software. Oracle has already developed and contributed an open source clustered file system to Linux. We expect to make additional contributions in the future."

PostgreSQL is not a standalone database product: it is distributed as a part of the PostgreSQL Global Development group. PostgreSQL's contractual relationship with Oracle comes up for renewal next year. Oracle fully expects to negotiate an extension of that relationship.

Terms of the transaction were not disclosed. But some says this is an hoax for April, 1st ;-)
